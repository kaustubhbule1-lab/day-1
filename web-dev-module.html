🌐 Web Development

7-day journey through HTML, CSS, and JavaScript — building from basics to interactive web projects.

Day 1 HTML Basics — Structure & Tags ▼

What I Learned

  • HTML Structure: DOCTYPE, html, head, and body tags
  • Headings & Paragraphs: h1–h6, p tags for text content
  • Links & Images: anchor tags with href, img with src and alt attributes
  • Lists: Ordered (ol) and unordered (ul) lists with li items
  • Tables: Creating structured data with table, tr, th, td tags
  • Semantic HTML: Importance of meaningful tag usage
Day 2 HTML Forms — User Input Elements ▼

What I Learned

  • Form Elements: input, textarea, select, button tags
  • Input Types: text, password, email, date, number, file
  • Radio & Checkbox: Single and multiple selection controls
  • Dropdown Menu: select with multiple option elements
  • Form Validation: required attribute and basic input constraints
  • Labels: Associating labels with input fields for accessibility
Day 3 CSS Basics — Styling & Selectors ▼

What I Learned

  • CSS Selectors: Element, Class (.class), and ID (#id) selectors
  • Colors & Backgrounds: Applying colors using names, hex, and rgb values
  • Typography: font-family, font-size, font-weight, text-align
  • Box Model: margin, padding, border properties
  • Specificity: Understanding which CSS rules take priority
  • Inline vs External CSS: Different ways to apply styles
Day 4 CSS Layout — Flexbox & Positioning ▼

What I Learned

  • Flexbox Container: display:flex, flex-direction, justify-content, align-items
  • Flex Items: flex-grow, flex-shrink, flex-basis properties
  • Space Distribution: space-between, space-around, space-evenly
  • Wrapping: flex-wrap for responsive row/column layouts
  • Alignment: Centering elements both horizontally and vertically
  • Practical Layout: Built multi-column box layout with flexbox
Day 5 HTML Project — Virat Kohli Documentary Page ▼

What I Learned

  • Page Structure: Building a multi-section webpage from scratch
  • Navigation Bar: Creating anchor-based navigation links
  • Content Layout: Organizing intro, career highlights, and image sections
  • Applying Styles: Background color, font styling, and margin/padding
  • Embedding Images: Using img tags with proper sizing and positioning
  • Project Practice: Combining HTML and CSS knowledge in a real page
Day 6 JavaScript Basics — Variables & Data Types ▼

What I Learned

  • Variables: var, let, const declarations and their differences
  • Data Types: String, Number, Boolean — how JS handles them
  • String Operations: Concatenation, type coercion with + operator
  • Boolean Arithmetic: true/false addition, subtraction, multiplication
  • typeof Operator: Checking the data type of a variable
  • console.log: Debugging and printing output to the browser console
Day 7 JavaScript Project — To-Do List App ▼

What I Learned

  • DOM Manipulation: Selecting and modifying HTML elements with JS
  • Event Listeners: Adding click events to buttons dynamically
  • Dynamic Content: Creating and appending new list items to the DOM
  • Input Handling: Reading user input from text fields
  • Project Integration: Combining HTML, CSS, and JS into a working app
  • User Experience: Building an interactive, functional web application
